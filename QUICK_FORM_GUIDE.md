## ðŸ†˜ Common Questions

**Q: Can I edit a form link after creating it?**
A: No, but you can deactivate it and create a new one.

**Q: What happens if a client submits with an existing Aadhar?**
A: The system will reject it and show an error.

**Q: Can clients save and continue later?**
A: No, they must complete the form in one session.

**Q: How do I know when someone submits?**
A: Check the Submissions page regularly. The count updates automatically.

**Q: Can I delete a submission?**
A: Currently no, but you can reject it to mark it as declined.

**Q: What file types can clients upload?**
A: JPG, PNG, and PDF files up to 5MB.
